## 常见问题

### "一台边缘设备" 是什么？

一台跑着 mqtt-mcp-server 的树莓派 / 香橙派 / 工控机 = 一个节点。比如 3 个车间各装一台 = 3 个节点。

### 为什么买断而不是订阅？

中国客户不喜欢月付。一次付费，终身使用，没有隐形费用。

### Pro 包含源码吗？

不包含。Pro 是闭源二进制发布。开源版完全开放，你可自由修改。

### 技术支持和更新？

- 开源版：GitHub Issues 社区支持
- Pro：QQ 群 VIP 通道 + 1 年免费更新
- Enterprise：专属技术群 + 3 年免费更新

### 怎么购买？

扫码加 QQ 群 → 联系群主 → 获取 License 文件 → 部署生效。
